- `#/nsyte.exe` appended to the URL renames the downloaded file inside Scoop's apps/ directory to `nsyte.exe` (so the binary lives at `%USERPROFILE%\scoop\apps\nsyte\current\nsyte.exe` regardless of release filename)

- `PLACEHOLDER_VERSION`         — bare semver in the `version` field AND in the literal URL string (Phase 24 CI will replace it)

Scoop autoupdate uses `$version` — this is a Scoop runtime variable, NOT a placeholder. Do not replace it. Scoop's checkver detects the latest tag, strips the leading `v`, and substitutes `$version` automatically.
