<action>
Create `src/lib/nip5a.ts` with the following exports:

1. **`encodePubkeyBase36(pubkey: Uint8Array): string`**
   - Convert 32-byte Uint8Array to BigInt by iterating bytes: `num = (num << 8n) | BigInt(byte)`
   - Call `num.toString(36)` to get lowercase base36 string
   - Left-pad with `'0'` to exactly 50 characters: `.padStart(50, "0")`
   - Throw Error if input is not exactly 32 bytes: `"Expected 32-byte pubkey, got {n} bytes"`
   - Return the 50-character lowercase string

2. **`decodePubkeyBase36(encoded: string): Uint8Array`**
   - Validate input is exactly 50 characters: throw `"Expected 50-character base36 string, got {n} characters"` if not
   - Validate all characters match `[0-9a-z]`: throw `"Invalid base36 character: '{char}'"` if not
   - Parse base36 string to BigInt: iterate chars, `num = num * 36n + BigInt(parseInt(char, 36))`
   - Convert BigInt to 32-byte Uint8Array: extract bytes with `num & 0xFFn` and `num >> 8n`, then reverse
   - Return the 32-byte Uint8Array

3. **Constants:**
   - `export const PUBKEY_B36_LENGTH = 50;`
   - `export const PUBKEY_BYTE_LENGTH = 32;`

File structure:
```typescript

export const PUBKEY_B36_LENGTH = 50;
```

<read_first>

</read_first>
<action>
Add to `src/lib/nip5a.ts` the following exports:

1. **`validateDTag(tag: string): { valid: boolean; error?: string; suggestion?: string }`**
   - If empty string: `{ valid: false, error: "Site identifier cannot be empty" }`
   - If fails `^[a-z0-9-]{1,13}$` regex: build specific error message and suggestion
   - If ends with `-`: `{ valid: false, error: "Site identifier cannot end with a hyphen", suggestion: tag without trailing hyphens }`
   - If valid: `{ valid: true }`

2. **`suggestIdentifier(input: string): string`**
   - Transform input to a valid dTag suggestion:
     - Convert to lowercase: `.toLowerCase()`
     - Replace underscores with hyphens: `.replace(/_/g, "-")`
     - Strip all characters not matching `[a-z0-9-]`: `.replace(/[^a-z0-9-]/g, "")`
     - Remove trailing hyphens: `.replace(/-+$/, "")`
     - Truncate to 13 characters: `.slice(0, 13)`
     - If result ends with hyphen after truncation, remove trailing hyphens again
     - If empty after all transforms, return `"site"`

Error message format for validateDTag when suggestion differs from input:
`Invalid site identifier "${tag}". Try "${suggestion}" (lowercase, no underscores, max 13 chars)`

When input is too long but otherwise valid-looking:
`Site identifier "${tag}" is too long (${tag.length} chars, max 13). Try "${suggestion}"`

When input has trailing hyphen only:
`Site identifier cannot end with a hyphen. Try "${suggestion}"`
</action>
