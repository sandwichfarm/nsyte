# Phase 26 Plan 01: Nix Flake Update Pipeline

## Objective

Replace the `publish-nix` stub with a release job that converts setup SHA256
outputs to Nix SRI hashes, patches `flake.nix`, validates the flake with Nix,
logs Linux ELF linkage, and commits the updated flake state back to the default
branch.

## Tasks

1. Install Nix in the `publish-nix` job using `cachix/install-nix-action@v25`.
2. Convert setup job hex SHA256 outputs with `nix hash to-sri --type sha256`.
3. Patch `PLACEHOLDER_VERSION` and the three active SRI placeholders in
   `flake.nix`.
4. Log `file nsyte-linux-<VERSION>` output for ELF linkage visibility.
5. Run `nix flake lock` and `nix flake check --no-build`.
6. Commit `flake.nix` and `flake.lock` when changed, then push to the default
   branch with `RELEASE_TOKEN`.
